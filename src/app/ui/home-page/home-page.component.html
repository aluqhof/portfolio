My Projects

Student Go

Student Go is a comprehensive application designed to simplify student life and event management. Developed using Flutter for the mobile app, Angular for the web interface, and Spring Boot for the backend, Student Go offers a seamless experience across platforms. To manage the database, PostgreSQL is utilized, and Docker is used to set up and run the database environment. With features including event ticket purchasing, interactive maps, calendars, and QR ticketing, along with a robust admin platform for organizers to manage events and entities, Student Go provides a complete solution for students and event organizers alike.

CD Pista Central

CD Pista Central is a comprehensive club management system for court reservations, developed using Spring Boot, Thymeleaf, and JavaScript. Designed to streamline the booking process, registered members can easily reserve courts and receive confirmation via email. Administrators have access to a powerful panel to manage all aspects, including viewing reservations in a table and calendar format for each court. They can also manage members by adding, removing, and editing their details, as well as adding new sports and courts. This system ensures a smooth experience for both members and administrators.
